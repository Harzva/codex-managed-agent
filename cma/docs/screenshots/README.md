# Screenshot Workspace

This directory is the canonical home for Marketplace and README screenshots.

## Intended Files

Required:

- `main-dashboard.png`
- `board-active-cards.png`
- `needs-human-dock.png`
- `docked-layout.png`

Optional:

- `insights-topic-map.png`
- `inspector-detail.png`

## Workflow

1. Read `../../SCREENSHOT_INVENTORY.md`
2. Follow `./CAPTURE_CHECKLIST.md`
3. Use `./RESOURCE_PLAN.md` to prepare the right UI state before capturing
4. Save final images here with the exact filenames listed above
5. Replace README placeholder paths with the final images if needed

## Rule

Do not store rough screenshots, drafts, or cropped experiments here long-term.
Keep only the final images used in README / Marketplace materials.
