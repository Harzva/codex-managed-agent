# No Editor First Workflows Task Plan

Status: complete

## Priority

Priority 2

## Objective

Make the managed center the default place for routine agent operations.

## Scope

- send prompt from cards
- loop control from cards
- inspect without opening extra surfaces
- log/result visibility inside cards and drawer

## Tasks

### Task 1 — Card-Level Operations

- [x] keep prompt send, Codex jump, and loop actions usable from board cards
- [x] reduce unnecessary editor or terminal jumps

### Task 2 — Embedded Status and Logs

- [x] show enough runtime/log context inside cards and drawer
- [x] avoid forcing users to open log files for common checks

### Task 3 — Controlled Fallbacks

- [x] use editor/terminal only when needed
- [x] make fallback actions explicit rather than surprising

## Constraints

This is a design target, not an absolute rule. Some actions will still need external surfaces.

## Exit Criteria

- common workflows stay inside the managed center
- editor and terminal become fallback tools, not the default control path
